# hadoop 命令

### 1. 查看文件

```text
hadoop fs -ls hadoop文件路径
```

### 2. 文件大小

```text
hadoop fs -du hadoop文件路径
```

### 3. 下载文件

```text
hadoop fs -get hadoop文件路径 本地文件路径
```

### 4. 上传文件

```text
hadoop fs -put 本地文件路径 hadoop文件路径
```

## 5. 删除文件

```text
hadoop fs -rm hadoop文件路径
```
