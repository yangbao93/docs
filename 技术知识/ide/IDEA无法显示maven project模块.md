# IDEA无法显示maven project模块

遇见idea右侧maven project无法显示的问题，解决方式： 1. 打开setting --&gt; 2. 选择plugins--&gt; 3. 选择maven--&gt; 4. 看两个是否都选上了，如果没有则都勾选；如果都勾选就取消以后再勾选--&gt; 5. 重启IDEA即可
